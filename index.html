CFDScript

CFDScript is an open-source JavaScript library for solving Partial Differential Equations using the Finite Element Method. It enables the creation and running of browser-based simulations for physics and engineering problems with visualization capabilities. Its design goal is to enable performing simulations on-the-fly, without the need to install heavy software or any other prerequisites.

Download

CFDScript is currently under development with continuous addition of new features and improvements. The latest version of CFDScript can be downloaded from our GitHub repository.
